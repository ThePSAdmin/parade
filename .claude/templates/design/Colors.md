## Usage Guidelines

### Accessibility
- Ensure minimum contrast ratio of 4.5:1 for normal text
- Use 3:1 minimum for large text and UI components
- Test all color combinations with accessibility tools

### Implementation Notes
- Define colors as CSS custom properties or design tokens
- Use semantic names (e.g., `--color-error`) not literal names (e.g., `--color-red`)
- Support both light and dark modes via prefers-color-scheme
