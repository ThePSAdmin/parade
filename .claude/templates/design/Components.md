## Buttons

### Primary Button
- **Background**: Primary color (`#007AFF`)
- **Text**: White (`#FFFFFF`)
- **Padding**: 12px 24px
- **Border Radius**: 8px
- **Font**: Body size, Semibold
- **States**:
  - Hover: 10% darker background
  - Active: 20% darker background
  - Disabled: 50% opacity, no pointer events
  - Loading: Show spinner, disable interaction

### Secondary Button
- **Background**: Transparent
- **Border**: 1px solid Primary color
- **Text**: Primary color
- **Padding**: 12px 24px
- **Border Radius**: 8px
- **States**: Same opacity/darkness rules as Primary

### Tertiary Button
- **Background**: Transparent
- **Border**: None
- **Text**: Primary color
- **Padding**: 8px 16px
- **States**: Hover shows subtle background tint

### Destructive Button
- **Background**: Error color (`#FF3B30`)
- **Text**: White
- **Use for**: Delete, remove, cancel actions
- **Require confirmation for destructive operations**

## Cards

### Standard Card
- **Background**: Surface color (`#F2F2F7` / `#1C1C1E`)
- **Border Radius**: 12px
- **Padding**: 16px
- **Shadow**: `0 2px 8px rgba(0, 0, 0, 0.1)`
- **Spacing**: 16px gap between cards

### Interactive Card
- **Cursor**: Pointer
- **Hover**: Subtle lift effect (`translateY(-2px)`)
- **Active**: Slight scale (`scale(0.98)`)
- **Transition**: 150ms ease-out

### Grouped List

## Feedback

### Toast/Snackbar
- **Position**: Bottom center, above tab bar
- **Background**: Dark (`#1C1C1E`) with high opacity
- **Text**: White, Body Small
- **Duration**: 3-4 seconds, dismissible
- **Actions**: Optional text button

### Loading States
- **Spinner**: System activity indicator
- **Skeleton**: Animated placeholder shapes
- **Progress**: Determinate bar for known durations

### Empty States
- **Icon**: Relevant illustration or icon (64px)
- **Title**: H3 style, centered
- **Description**: Body style, Text Secondary
- **Action**: Primary button for next step

## Animation

### Timing
- **Fast**: 150ms (micro-interactions)
- **Normal**: 250ms (transitions)
- **Slow**: 400ms (major changes)

### Easing
- **Default**: `ease-out` (cubic-bezier(0, 0, 0.2, 1))
- **Enter**: `ease-out`
- **Exit**: `ease-in`
- **Bounce**: `cubic-bezier(0.68, -0.55, 0.265, 1.55)`
